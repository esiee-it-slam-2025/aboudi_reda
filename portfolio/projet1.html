Projet : Yueii

Description du projet

Ce projet consiste à un jeu de programmation basé sur Python. Le but est de manipuler les bases du langage tout en développant un projet complet.

Technologies et Outils

  • Langage : Python
  • IDE : Visual Studio Code
  • Plateforme : GitHub pour la gestion des ressources

Bilan de Compétences

Ce projet m'a permis d'approfondir mes connaissances en Python, notamment sur les aspects suivants :

  • Manipulation des boucles et conditions.
  • Gestion des fichiers et structures de données simples.
  • Utilisation de GitHub pour versionner et partager mon code.

Ce projet m'a également appris à documenter mon travail et à expliquer mon code.
